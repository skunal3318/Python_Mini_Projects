📌 Prerequisites:
  • Python 3.x installed

📥 Clone the repo:
  git clone https://github.com/skunal3318/Python_Mini_Projects.git

📂 Navigate and install dependencies (if any):
  cd Python_Mini_Projects
  pip install -r requirements.txt

▶️ Run a project:
  python script_name.py
